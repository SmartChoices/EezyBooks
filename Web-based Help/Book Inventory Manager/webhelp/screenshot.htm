Sending us a "screen shot"

We ask that you do all of your problem reporting by using the Support Center ticket tracking system found here.  This will insure that nothing gets lost, and you can continually track the progress of your issue.  Please fill in all of the information, such as Operating System, Service Pack level and other information that will enable us to reproduce the issue.

Occassionally, you might have an issue where we ask you to send a screen shot of the offending error message or some other window.  This is how you do it...

Installing a Screen Shot Program

Go here and download ScreenPrint32, an excellent FREE screen shot utility.  Install it and start it  (the following examples use the program ScreenPrint32).

 

Capturing the screen

When you start the program, the first thing you will see is the the screen shown in Figure 59a below.

Figure 59a.  Click on the option pointed to by the red arrow.

Next, you want to click on the Save Options tab and click on the two boxes as indicated by Figure 59b below.

Figure 59b.  Save Options tab

Once you have done both of these tasks, click OK and the ScreenPrint32 setup screen goes away.

Doing the actual capture 

To capture a window or portion of your screen, press your left mouse button and drag it over the area you want to capture.  Release the button and press the right mouse button to complete the capture.  Your captured image will be found in the directory you selected in Folder (see Figure 59b).  Take this file and send it to us as an attachment using the Ticket Tracking system.
