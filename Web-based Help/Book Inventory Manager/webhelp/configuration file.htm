The Configuration file

We use a configuration file to give the program information it needs when starting up that cannot be obtained by other means.  It is named Inventory.cfg and is found in the C:\Prager directory.  You can use a normal text editor (Notepad works nicely) to modifyt it if necessary (the installation program will do this for you except in difficult circumstances, such as a change in installation directory, etc).  The contents look like this:

<Configuration>
    <DatabasePath>C:\Prager\</DatabasePath>
    <BackupPath>c:\Prager\Backup\</BackupPath>  <!--  in which directory do you want your backup files placed? -->
    <ExportPath>c:\Prager\Export\</ExportPath>  <!--  in which directory do you want your export files placed? -->
    <DaysRetention>30</DaysRetention>  <!--  how many days do you want to keep backup and export files?  -->
</Configuration>

The lines are self-explanatory.  Note that 30 days retention assumes you backup your database daily.  You can extend this if necessary; it is only used to "prune" the directory so as to avoid having thousands of unused files taking up space.
