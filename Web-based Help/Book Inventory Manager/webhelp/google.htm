Google Merchant Center

There are several things that must be accomplished before you can upload your inventory to Google Merchant Center.  You must register, set your options in the Inventory Program and then you are able to automatically upload with 1-click to Google Merchant Center.

1.  Google Setup

  • If you don't already have a Google account, go here to create one.

2.  Registering with Google

Registering with Google Merchant Center is a five (5) step process.  They must be done in order, and where indicated by Google, you must wait for a response before continuing to the next step.  So go to the Google Merchant Center - Bulk Upload Instructions page and follow the directions.  The following notes may help:

  1. Item type should be set to "Products" and the Data Feed name should be the same as what you have in Registered File Name above.  Make sure you don't delete this data feed on the Google site, otherwise you will have to register it all over again, which is not a big deal, but would have to be done.
  2. Don't forget to set up a FTP Account (without this, we can't upload to Google Merchant Center) by going to the Settings page, and click on the FTP acccount tab.
  3. Next, you have to manually upload one (and only one book!) book to Google Merchant Center, which will register your "data feed" (the file we send to Google Merchant Center).
  4. For "creating the Bulk Upload", take one or two of your books and "update" them so they will show up in the "files waiting to be exported" count.  Make sure that you have the User ID and Password set correctly on the User IDs and Password page in the Inventory program.  Then do an upload to Google Merchant Center.  Make sure you have checked Google Merchant Center on the upload page.
  5. Once this is done, you will receive an email from Google Merchant Center stating that your file has been accepted.  If Google does not accept your file, and you can't figure out what you did wrong, please notify us, and we will try to help you through this.  It's a confusing process until you have done it once, and the Google staff is wonderful in helping you fix any of your problems.

 

3.  Normal uploading to Google

Once you have registered, and Google has accepted your file, you need to place your User ID and Password in the program, by going to the User ID's and Passwords page and entering them.  This is the User ID and Password you created on the FTP Account page.  Don't forget to save them!
