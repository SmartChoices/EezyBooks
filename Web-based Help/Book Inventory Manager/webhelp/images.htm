Images

First of all, the actual image is not stored in the database; the location of the image on your machine is what is stored in the database.  So, if at a later time you decide to move your images to another folder, you will have to update each record in the database with the new image location.  For this reason, we suggest you make a new directory where you have the Backup and Export directories (folders). 

And at the present time, the actual images are not uploaded to the listing services because there is no easy way to do this.  Very few of the listing services accept images, and there is no common way to upload them; each service is different.  We are working on a solution to this problem.  Basically, images are for your use.  If you want to upload them, you have to create them in a format that each listing service will accept and then manually upload them. 

Adding or Changing Images

To Add or Change an image for this selected record (you must select a record that this image will belong to), click on Select Image, and a Open Dialog (that's what we call it) will appear..

Change directories until you have the directory where your images are stored and select one for this book's record.

Figure 60c.  Once you have clicked the Open button (as shown in Figure 60b above), you will see your image in the image box for that record.  The Update Book button will turn red, telling you that if you want to save this image filename with the record, you must click Update.  The image filename is now attached and stored with this record in the database.

Deleting Images

To delete an image click delete.  There is no need to click Update because the deletion is done automatically.
