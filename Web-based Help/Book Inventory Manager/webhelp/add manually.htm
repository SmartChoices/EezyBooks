Adding a book manually

 

Check Options

When adding books to the database, you have limited control of the SKU (Stock Unit, a unique number used to identify this book) field.  This is the key in the database, which must be unique.  There are two ways you can enter this information:

  1. On the Program Options page (see figure above), there is an option for Automatic SKU Numbering.  When this option is selected (as evidenced by the check by the Auto Numbering field) the SKU field  on the Book Detail page is disabled (greyed-out) and the number is automatically generated by adding 1 to the last number on file.  This field is updated when the Add button is clicked. Note:  when using Auto-Numbering, the number will automatically be expanded to 10 digits by padding on the left with zeros.  NOTE:  Unless you want to start with the number 1, you should enter your choice for the starting number and click Add  before clicking Auto Numbering.
  2. If you choose to enter your own SKU, do not check Auto Numbering, in which case the field is enabled to allow you to enter the SKU.  Remember, this number must be unique.
  3. A new feature is how multiple copies are handled.  You can either have one SKU for each copy, or you can have all copies under one SKU.  The secret is in the quantity field;  When you have one SKU for each book and if the quantity field is zero, the book is considered sold.  If you have one SKU for multiple copies of a book, and you sell one of the copies, you must decrement your copy count.  The program will not do it for you, since it does not know how many copies you have sold.  When the quantity count goes to zero, the book is then marked sold which indicates no more copies are available.

 

Adding the book when you have an ISBN...

To add a book manually for when you have the ISBN requires the following steps:

  1. Click on the Clear button to clear the Detail panel of any existing prior data (failure to do this will result in the prior data being carried forward to the current book being added).  Notice the yellow highlighting on some of the input entries.  This indicates the minimum amount of information that is required for entering a book into the database.
  2. Enter the ISBN in the text box and click on Get Book Info (with ISBN).  If you have a barcode scanner, just click on the textbox for ISBN and scan the book's ISBN with your scanner.  The ISBN will automatically be decoded and placed in the ISBN textbox. This requires an internet connection for the program to be able to find the book information on the internet.  It takes a few seconds for the data to appear, automatically being populated in the Title, Author, Publisher text boxes depending on what was found.  If the Auto Pricing Lookup Option was checked, any pricing information is displayed in the Pricing Results task panel.   Now, enter the data, checking the appropriate boxes for Binding, Condition, etc. into the appropriate spots in the Detail Panel.  If you don't like the default selections in Binding, Condition, etc., you can enter your own data in the selected box rather than using the down arrow.  The drop-down boxes for Binding, Condition, Jacket, Type and Edition allow you to enter your own text if you don't like what you see in the list.  But, if you don't use the supplied condition, some data being sent to some of the listing services (namely Amazon.com) will not be translated correctly (you will be warned).  
  3. If you want to have canned text, which makes it more easier to add repetitive text to the description field, go to the Canned Text tab, fill out the title and text and then click on the buttons.  You can have up to five different canned texts.  The limit of the Description is 500 characters.
  4. To enter this book into a catalog, press the Select Catalog button next to the Catalog entry box;  the Catalog tab will open, where you can select a catalog.  The selection you make will be populated to the Catalog entry automatically.  Do not try to add catalog entries in this box; you must go to the Catalog Task tab (also where you can find the information about sub-catalogs) to make any additions or deletions to the catalog file. 
  5. Private Notes do not get exported to the listing services, they only appear in the database.
  6. If you didn't check Auto Pricing Lookup in the Program Options, you can still get prices from the internet.  Once you have the information entered, you can automatically pass this to the pricing routines by clicking on the Get Prices from Internet button.  This will automatically go to the internet for pricing as seen in the figure below.

  7. Finally, when you are satisfied all of the data is correct, simply press the Add button.  Enter the total number of copies you have to add in the window next to the Add button and it will clone that many of copies if you have checked Multiple Copies > Multiple SKU's on the Program Options page, in which case each copy will have it's own SKU.

 

Adding the book when you don't have an ISBN...

To add a book manually for when you don't have the ISBN requires the following steps:

  1. Click on the Clear button to clear the Detail panel of prior data (failure to do this will result in the prior data being carried forward to the current book being added).  Notice the yellow highlighting on some of the input entries.  This indicates the minimum amount of information that is required for entering a book into the database.
  2. Enter the Title and Author in the spaces provided, along with any of the other information that you have for this book.  Once you have finished entering the book information (as indicated above as if you had an ISBN), you can get the pricing information by clicking Get Info (w/o ISBN) which will transfer the Title and Author to Biblio.com to find the pricing information for that book.  Scroll through the information returned to establish a starting price for the book.  If you wish to use another web site other than Biblio.com, you can select that site by clicking on the button for that site.  Biblio.com is the default at this time, buy you can also choose AddAll.com or Bookfinder.com.  Just enter the book information in the spaces provided as you would normally do with that website, and press Enter on your keyboard.  (This tab acts like a mini web browser window so you can do almost everything in this tab that you could do with a full size browser.)


The scoop about Shipping...

As you can see, things have changed with respect to shipping, thanks to Amazon.com' new requirements.  If you just want the normal shipping, then click on Domestic Standard.  If you offer expedited domestic shipping, then click either Domestic Expedited, Two Day or One Day Domestic.  You will notice that the later two are for Amazon only, and are not exported to other venues.  International Standard and Expedited are also available for all venues.



Relisting a book

There is also a Relist button which will only work when the book's status is Sold and the quantity is zero (0); therefore, do not change the quantity to a positive number, just click the Relist button.  It's function is to allow the book to be relisted if it is returned, etc.  The book will have a new SKU (which is the key into the database) and will be marked For Sale.  Otherwise, it is an exact duplicate of the book that was sold but being returned. 

Getting current pricing information...

Once you have added books, the price you have entered can be compared against what the competition is selling the same book for.  Use the Re-pricing Tool tab to start the re-pricing routine after you have entered all of your books for the day.  This will bring your prices more inline with what is current.

Amazon volume information

When using the Volume field, enter a single number or identifier only. To list a complete set, continue to use the title field to indicate the total number of volumes in the set. To list multiple volumes from an incomplete set, list each book individually as a single volume.
