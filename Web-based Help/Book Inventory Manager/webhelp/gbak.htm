Sending us your backup file

There are times we are going to ask for you to send us a backup file... here is how you do that:

  1. FInd the latest file that ends with .gbk in the c:\Prager\Backup directory.
  2. Add that file as an attachment to either a new email (sent to support [at] pragersoftware [dot] com) or add it as an attachment to the ticket (found at pragersoftware [dot] com/ostic

That's all there is to it... we will take the backup file and attempt to recreate your issue.  We will notify you of the status of the issue after we have tried to reproduce the issue.
