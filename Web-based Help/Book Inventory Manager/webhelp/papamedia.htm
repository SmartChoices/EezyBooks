Setup for Papa Media venue

General Information

Papa Media is a listing venue that only charges for books sold, not for listing on their site.  Therefore, there is much interest in how we interact with the Papa Media site.  For starters, uploading to the Papa Media site is a bit different, in that they "pull" the files we upload to our server.  This is different from most of the other sites in that it does NOT go directly to the Papa Media site - they visit our server several times a day to get the files that users have uploaded.

  • Once you have all of your inventory in the Prager Inventory program, the next thing you have to do is put your email address on the User ID and Password page, next to the Papa Media line.  There is no password required.
  • Next, go to the Export page and create a series of export files;  there will be several types of files, UIEE, Tab-delimited, HomeBase,etc.  You will not see one for Papa Media because we have created it under a different name, and the program will automatically find the correct file to export.
  • Finally, go to the Upload page (you should be taken there automatically upon completion of the Export) and click on the checkbox next to Papa Media.  Then click on the Upload button.
  • You can watch the status of the upload process in the yellow window.  If there are any errors that you don't understand, contact us.
