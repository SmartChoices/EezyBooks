Using the ASIN lookup tool

This program has the ability to do a lookup to Amazon.com for ASINs when you don't have a ISBN.  (an ISBN is preferred, because other listing venues do not recognize Amazon's ASIN).

You would start by selecting the ASIN page (tab), which will cause the program to go through your database looking for all books that don't have either an ISBN or an ASIN.  It then lists what it found in the top portion of the page; all you have to do is click on one of the books and the program does everything else.  Once you have the list, select the book closest to yours and click Update.  That's it!  If you have too many books to do at one sitting, don't worry.  Stop anywhere, and the next time you go to the page, all of the books that still don't have ASINs or ISBNs will be listed, ready for you to start.
