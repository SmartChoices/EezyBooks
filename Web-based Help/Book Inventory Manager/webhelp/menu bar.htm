Menu Bar

Database Menu Item

The Database Menu Item consists of two selections.  They are as follows:

  • Backup - causes the entire database to be saved
  • Restore - causes the entire database to be restored from a previous backup
  • Exit - exit the program

 

Tools Menu Item

The Tools Menu Item consists of:

  • A tool to remove the shipping message in the Description portion of a record
  • A tool to work offline - good when you don't have an internet connection available
  • Initialize the Quantity field in the database (major changes were made in version 11.5.0 - this corrects any errors in the conversion
  • ASIN Update - allows updating the ASIN of your books with a file received from Amazon.com

 

Help Menu Item

 

The Help Menu Item is composed of:

  • Help - will bring up the web-based Help file for this program.  You may use the F1 key to accomplish the same thing.
  • Tutorials - web-based tutorials on most of the major functions you might need help with
  • Purchase/Renew license - will take you to our web site where you can purchase or renew a license for this program.
  • Get Updates - by clicking on this item, the program will go to the web site and check for the existence of a newer version of this program other than the one currently running.  It will not download the newer version, but alert you to the fact that a newer version exists.  Please note that no data is transmitted from your machine to the website; the website is only interrogated for the latest version of the Inventory Program and that information is returned to the Inventory Program running on your machine.  This is used only if you don't have it set in Program Options, which is recommended.
  • Website - will take you to the home page on our website.
  • Report a Problem - a link to the web site support page to enable you to file an issue on our Ticket Tracking system.
  • About - displays the current version number and other pertinent information
