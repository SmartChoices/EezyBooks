Catalogs

Catalogs are methods to segregate books by subject, such as History, Cooking, etc.  They come in two flavors:

  • Amazon.com
  • non-Amazon.com

Amazon.com catalogs are prefixed by a 6 digit number (most of the time), where non-Amazon.com catalogs are not.  Your choice of which catalog to use is predicated on whether or not you upload to Amazon.com or not.  Non-Amazon sites will accept the catalog entries without the prefix.  The figure below shows the Amazon.com catalog structure.

In addition, for either catalog, you may have sub-catalogs (this makes more sense for where you use the non-Amazon.com catalog).  It allows you to further breakdown your catalogs, from general to specific.   If you want to use both catalogs, you must indicate that by clicking on the Use Primary and Secondary Catalogs in the Program Options page (tab).

 

 

Creating the Catalogs

Catalogs are created in one of two ways:

  • By importing from a file
  • By manually entering each catalog genre

To import from a text file, do the following:

  • create a text file with one entry on each line, saving the file as a .txt file
  • click on the Browse button  and select the file you have just created
  • if you want to delete the current contents, click that box; otherwise, you will append the entries to the current catalog entries
  • click the Import button

To import manually, do the following:

  • in the space to the left of the Add Entry button, add the name of the catalog entry exactly as you wish it to appear
  • click the round button for either the Primary Catalog or Secondary Catalog; this is where your entry will be placed.  NOTE: you must have selected a primary catalog entry (it must be indicated as selected, normally by turning blue) before you can add a secondary catalog entry.
  • Click the Add Entry button

To delete a catalog entry, do the following:

  • Select either a Primary Catalog or Secondary Catalog entry for deletiion (it must be selected as indicated by the dark blue color)
  • click the Delete button (the program knows which catalog to delete the entry from)

 

Using the Catalogs

In the Book Detail page (tab) you will find the Select Catalog button as shown above.  Do not enter anything in the catalog entry unless you are making a special one-time entry to the book's catalog.  Just click the Select Catalog button and you will be transferred to the Catalog page (tab) where you can select a Primary Catalog entry.  If you have not entered a secondary catalog entry, you will  be returned to the Book Detail page.  If you want to enter a Secondary catalog entry for this book, you must do it before you click on the Select Catalog button (this is the only way the program knows not to return you to the Book Detail page).  If there are Secondary Catalog entries attached to the Primary Catalog, they will be displayed.  At this time, you may choose one or not.  If you decide not to choose a secondary catalog entry, then manually click on the Book Detail page to return and continue providing data on the current book.

Catalog entries may be changed at any time by selecting the book and clicking on the Select Catalog button.
