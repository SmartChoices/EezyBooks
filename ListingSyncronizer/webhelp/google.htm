Uploading to Google Base

There are several things that must be accomplished before you can upload your inventory to Google Base.  You must register, set your options in the Inventory Program and then you are able to automatically upload with 1-click to Google Base.

1.  Google Setup

  • If you don't already have a Google account, go here to create one.
  • Next, open the Inventory program and go to Main Menu -> Tools -> Google Setup
  • Fill in the following information:
    • Google User ID - enter the Google ID you received from step 1 above.
    • Registered File Name:  pick any name for a filename; this name will be used during the upload process.  Google Base requires this non-changing filename to enable files to be uploaded.
    • Bookstore Name - enter the name of your bookstore.   This name will be carried in the file transmitted to Google.
    • You must have a URL that points to your shopping cart web page, unless you are using Google Checkout.
    • Payment types accepted - click all those that apply.  See this Google Help page for further information.
    • Sales Tax Info - enter appropriate tax info
    • Shipping Information - enter your shipping charges for each of the appropriate shipping types.  You do not have to fill in any that you don't use.
    • Save the information and close the window.

2.  Registering with Google

Registering with Google Base is a five (5) step process.  They must be done in order, and where indicated by Google, you must wait for a response before continuing to the next step.  So go to the Google Base - Bulk Upload Instructions page and follow the directions.  The following notes may help:

  1. Item type should be set to "Products" and the Data Feed name should be the same as what you have in Registered File Name above.  Make sure you don't delete this data feed on the Google site, otherwise you will have to register it all over again, which is not a big deal, but would have to be done.
  2. Don't forget to set up a FTP Account (without this, we can't upload to Google Base) by going to the Settings page, and click on the FTP acccount tab.
  3. Next, you have to manually upload one (and only one book!) book to Google Base, which will register your "data feed" (the file we send to Google Base).
  4. For "creating the Bulk Upload", take one or two of your books and "update" them so they will show up in the "files waiting to be exported" count.  Make sure that you have the User ID and Password set correctly on the User IDs and Password page in the Inventory program.  Then do an upload to Google Base.  Make sure you have checked Google Base on the upload page.
  5. Once this is done, you will receive an email from Google Base stating that your file has been accepted.  If Google does not accept your file, and you can't figure out what you did wrong, please notify us, and we will try to help you through this.  It's a confusing process until you have done it once, and the Google staff is wonderful in helping you fix any of your problems.

 

3.  Normal uploading to Google

Once you have registered, and Google has accepted your file, you need to place your User ID and Password in the program, by going to the User ID's and Passwords page and entering them.  This is the User ID and Password you created on the FTP Account page.  Don't forget to save them!
