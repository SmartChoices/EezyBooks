The Configuration file

If you are running this program on Vista or Windows 7, read this carefully!

We use a configuration file to give the program information it needs when starting up that cannot be obtained by other means.  It is named Inventory.cfg and is found in the same directory as where you installed the program.  You can use a normal text editor (Notepad works nicely) to modifyt it if necessary (the installation program will do this for you except in difficult circumstances, such as a change in installation directory, etc).  The contents look like this:

<Configuration>

<!--  do you see the four characters at the beginning of this line?  and the 3 characters at the end of this line? -->
<!--  well, those mark the characters between them as a "comment" (both the start and the end)...  -->
<!--  so, remove those characters for whichever line for the FirebirdInstallationPath that pertains to your system.  -->
<!-- (you can right-click on the Computer icon (on desktop or Start menu), then left-click on Properties to see if you have a 64-bit machine)  -->

    <!-- uncomment the line below ONLY if you have a 64-bit machine (x64) AND are running Vista (x64) or Windows 7 (x64) and comment out the line for 32-bit machines-->
    <!-- <FirebirdInstallationPath>C:\Program Files (x86)\Firebird\bin</FirebirdInstallationPath> -->

    <!-- all other cases (32-bit machine or running a 32-bit version of Windows) use the line below, leaving the above "commented out"  -->
    <FirebirdInstallationPath>C:\Program Files\Firebird\bin</FirebirdInstallationPath>


    <!--  Prager.Options.xml file path if networked  (eg. \\192.168.0.248\Prager where the IP address points to c:\Prager on the host which must be shared with the name of "Prager")  -->
    <!--    <ProgramOptionsFilePath>\\192.168.0.248\Prager</ProgramOptionsFilePath>  -->
   

<DatabasePath>C:\Prager\</DatabasePath>

    <BackupPath>c:\Prager\Backup\</BackupPath>  <!--  in which directory do you want your backup files placed? -->
    <ExportPath>c:\Prager\Export\</ExportPath>  <!--  in which directory do you want your export files placed? -->
    <DaysRetention>30</DaysRetention>  <!--  how many days do you want to keep backup and export files?  -->

</Configuration>

FirebirdInstallationPath is the path where you installed the Firebird database program; for Windows XP it is usually c:\Program Files.  For Vista and Windows 7, it is normally c:\Program Files (x86).  The DatabasePath is the location of the Firebird database (not the Firebird program) normally in c:\Prager.  If you are networking, see the section on Network Connections.

The other lines are self-explanatory.  Note that 30 days retention assumes you backup your database daily.  You can extend this if necessary; it is only used to "prune" the directory so as to avoid having thousands of unused files taking up space.
