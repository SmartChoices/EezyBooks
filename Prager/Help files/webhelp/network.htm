Networking Setup

In the same directory (folder) that you have installed the Inventory program, you will find a file named Inventory.cfg.  Using any text editor (Notepad will work just fine), open this file.  It will initially look something like this:

<Configuration>
<!-- don't change the following two lines unless instructed to do so  -->
    <ServerInstance>(local)</ServerInstance>
    <ServerInstallationPath>C:\Program Files\Microsoft SQL Server\MSSQL.1\MSSQL\Data\</ServerInstallationPath>
    <ServerInstallationPath>C:\Program Files\Microsoft SQL Server\MSSQL.1\MSSQL\Data\</ServerInstallationPath>

    <DatabasePath>c:\Prager\</DatabasePath>
    <FirebirdInstallationPath>c:\Program Files\Firebird\bin\</FirebirdInstallationPath>
   
  <!--  the following three lines may be changed if needed  -->
    <BackupPath>j:\Prager\Backup\</BackupPath>  <!--  in which directory do you want your backup files placed? -->
    <ExportPath>j:\Prager\Export\</ExportPath>  <!--  in which directory do you want your export files placed? -->
    <DaysRetention>3</DaysRetention>  <!--  how many days do you want to keep backup and export files?  -->
</Configuration>

The line you want to change is <DatabasePath>.  Change that line to conform to the following:

For Example:  assume you have two machines:  machineA and machineB.  The database (dbBooks.fdb) resides on machineA.  Do not change the configuration above for machineA; you want to change the Inventory.cfg file on machineB to be something like this, so that it points to machineA:

  • <DatabasePath>machineA:C:\Prager\</DatabasePath>

After the machine name, place a colon (:), followed by the drive letter, another colon,  and directory where the database is located.  Do not put in the database name in, the program does that automatically.  Make sure you have the ending backslash.  You can find your machine names by looking in My Computer -> Properties -> Computer Name tab.

In addition, you have to make some changes to your Firewall program (be it Windows firewall or some other vendor).  For example, open the Windows Firewall by going to Control Panel -> Windows Firewall.  Go to Exceptions.  You are going to click on Add Port...

Next, you are going to add two (2) ports: 3050 UDP and 3050 TCP...

Use the Browse button to navigate to the directory shown, and click on the fbserver.exe file.  (that way, you won't make any spelling mistakes!).  That's all there is to it; you can check it out by turning on both machines, make a change on Machine A, and see if it shows up on Machine B.  If you have any problems,  open up a support ticket at http://www.pragersoftware.com/mantis .
