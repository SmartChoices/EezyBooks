Importing a file

Creating the file to import

Before you can import a file, you have to create it.  If your inventory is on HomeBase, do the following:

To create a file from HomeBase:

1. In the File menu, select [Import/Export] and then [Export]. The "Export" dialog box is displayed.

2. Select your options for the export. Do one of the following:
  • Select [Books] to send us your books records. If you select this option, you have the choice to upload only changes from the last time you uploaded, and to send all of your
  • book records or only selected catalogs.
  • Select [Wants] to send us your want records. If you select this option, you can further choose to send only your recent changes.
  • Select [Catalog Names] to export a list of your catalog names in tab- or comma delimited format.
  • Select [Clients] to export your client records in tab- or comma delimited format. You can also choose to send only records changed since the specified date.
  • Select [Invoices] to send your invoice records in tab- or comma delimited format. You can also choose to send only records changed since the specified date.

3. Click the [OK] button. The Export as dialog box is displayed.

4. Select options from the File name and Save as Type lists. HomeBase 2.3 automatically creates a name for your file based on the date.

5. Click the [Save] button.

If you're not using HomeBase, but another type of inventory program, see if they can export the inventory in tab-delimited or HomeBase format.  Then follow the directions below.  If you keep your inventory on an Excel spreadsheet, or can get a copy of your inventory in Excel format, do the following:

To create a file from Excel (or any other spreadsheet program):

1.  In Excel, go to Main Menu -> File -> Save As

2.  In the drop down list "Save As Format", choose "text (tab delimited)"

Let us know if you are having problems doing the initial import; you can send us the exported file from above, and we will do the import for you and return the fully functional database.  Part of our outstanding support!

Importing a HomeBase, UIEE or Tab-delimited format file

Importing a file is the first thing after installation that should be done, unless you are starting from the beginning or don't have a file in the correct format.  

 The steps to import are as follows:

  1. Use the Browse button to find the filename, which once selected appears in the text box to the left of the button.  We strongly suggest using the Browse button to avoid errors in spelling, etc.
  2. Select the format of the input data, either HomeBase, UIEE or Tab-delimited format. 
  3. If you want to delete all of the books currently in inventory (purge), for instance if you want to delete all of the books from the Sample File to Upload, then check the box "Delete all books first?".  You will be asked to confirm that you want to delete all of them.
  4. Click the Import button.

NOTE: records imported using a UIEE format that have a status of "Withdrawn" (XB code = 5) are not imported.

If you have records that have duplicate SKUs (Book Numbers), you have three choices as noted in the figure above.  The "create a new key..." option is the default.

That's all there is to it!  You will see each record number being processed, which indicates everything is progressing normally. However, since there is an index being created on the database, imports of a large number of records may be very slow.  This is normally a one time issue, so have patience.

Rejected records are indicated by SKU (book number) and reason.

 

Importing a tab-delimited format file

Importing a tab-delimited file is different from a HomeBase or UIEE file; when you have tab-delimited checked, the screen above in Figure 16 appears.  The panel at the top of Figure 16 shows the column titles that were found when the file was read.  You have to take these titles and match them up with the titles in the database (listed to the right of each textbox). 

Use the Windows drag-and-drop feature (left-click, then holding the left mouse button down, drag the title from the top to the appropriate text box below) to accomplish selection from the top box where your titles are listed and drag it into the appropriate text box below.

Do this for all of the column-titles you can, then click the "Continue" button.  Note:  fields with yellow backgrounds are reqired fields (not shown in the figure above).  When you are done, save the mapping for later use.
