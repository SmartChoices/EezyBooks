Setting the ID Tech Scanner to read ISBNs

The manual that comes with the ID Tech scanner is really scary, so let's see if I can help...

  1. connect your scanner 
  2.  go to page 8 and put the scanner on Reset and press the button on bottom of the scanner 
  3. go to page 47 and put the scanner on ISBN ON and press the button again... 
  4. go to page 19 and put the scanner on CR (not CR+LF) and press the button again 
  5. you should be good to go... 

By the way, it will read 10-digit ISBNs as well as 13-digit ISBNs with the above program.
