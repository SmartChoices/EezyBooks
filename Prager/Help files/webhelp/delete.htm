Deleting a book from the database

 

  The Database Panel

 

How to delete a book from your database

If you remember, selecting any row in the Database Panel (see Figure 14) will populate the Detail Panel as shown in Figure 15.  All of the fields in the record have been populated.  Now, verify that this book is the one to delete and click the Delete button.  If you have checked Options -> Verify Deletes, a message box will appear asking you to verify the action.  I strongly recommend using this option for obvious reasons!

Note that this does NOT delete the book(s) from the listing service!  It only deletes from your database; to delete from the listing service, you only have to mark the book as 'Sold'

In addition, you can do mass deletes.  Just select each book that you want to delete in the Database Panel and click Delete.  If you have Verify Deletions checked, you will be asked to verify the delete for each book.

Oh, by the way, once deleted, it's gone forever!   The only way to get it back is to restore the database from a previous backup.
